## File Descriptions

### Primary Artifacts

| File | Role | Size |
|------|------|------|
| `untitled-1.tex` | Single-file LaTeX monograph; contains the `filecontents*` block that auto-writes the class file, all section content, all TikZ diagrams, all tables, all mathematical models, the Python workspace-manager listing, the Bash compilation listing, and the inline reference list. | 1,064 lines |
| `custom-academic-report.cls` | Custom LaTeX document class loaded by `untitled-1.tex`. Declared as `custom-academic-report` class (2026). Based on `article` 11pt/A4. | 36 lines |
| `untitled-1.pdf` | The primary deliverable. 58-page double-spaced academic paper with coloured hyperlinks, 7 tables (5 booktabs-style and 2 bordered `\hline`-style), 6 TikZ/pgfplots figures, 10+ numbered equations, an appendix, and a 12-entry reference list. | — |

### Build Artifacts (auto-generated, not committed to primary repo)

| File | Generated By | Purpose |
|------|-------------|---------|
| `untitled-1.aux` | LuaLaTeX pass 1 & 2 | Stores `\label`/`\ref` mappings, citation keys, hyperref destinations |
| `untitled-1.toc` | LuaLaTeX pass 1 | Section number → page number entries consumed on pass 2 |
| `untitled-1.out` | hyperref package | PDF bookmark tree |
| `untitled-1.log` | LuaLaTeX | Full diagnostic log including package loads, font substitutions, warnings |
| `untitled-1.synctex.gz` | LuaLaTeX | Compressed SyncTeX map for editor ↔ PDF navigation |

## Custom Class Features (`custom-academic-report.cls`)

| Feature | Configuration |
|---------|--------------|
| Base class | `article`, 11pt, A4 paper |
| Margins | 2.5 cm on all sides (geometry) |
| Line spacing | Double spacing (setspace) |
| Page layout | `\raggedbottom` |
| Math support | amsmath, amsfonts, amssymb |
| Graphics | graphicx |
| Typography | microtype |
| Tables | booktabs |
| Citations | cite package |
| Code listings | listings (`\small\ttfamily`, frame=single, breaklines) |
| Hyperlinks | hyperref (colorlinks: link/cite/url = blue) |
| Vector diagrams | tikz + pgfplots (compat=1.18) |
| TikZ libraries | shapes.geometric, arrows.meta, positioning |
